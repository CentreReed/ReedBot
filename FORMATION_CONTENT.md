# Contenu Complet de la Formation Centre Reed

---

## Vue d'ensemble du flux live

Le bot utilise actuellement 2 parcours:

- `N1` via `/start_onboarding`
- formation complete via `/finish_onboarding` une fois le tuteur apparie

Il n'y a pas de parcours `N1A` separe dans le code live. L'appariement sert de condition d'acces a la formation complete.

---

### N2-03 - Quiz Formation 1 : La methode Reed

1. **Quel est l'objectif principal du Centre Reed dans l'accompagnement des eleves ?**
   - A. Rendre l'eleve autonome dans son apprentissage
   - B. Ameliorer les notes rapidement
   - C. Completer les devoirs le plus vite possible
   - D. Preparer uniquement les examens
   - **Bonne reponse : A**

2. **Selon la vision du Centre Reed, quel est l'objectif a long terme ?**
   - A. Augmenter la motivation scolaire
   - B. Developper les fonctions executives
   - C. Reussir les examens finaux
   - D. Completer les devoirs sans aide
   - **Bonne reponse : B**

3. **Quel element est considere comme la cle #1 de la reussite d'un eleve ?**
   - A. Le materiel pedagogique utilise
   - B. Le nombre d'heures de tutorat
   - C. La relation de confiance avec le tuteur
   - D. La difficulte des exercices
   - **Bonne reponse : C**

4. **Dans le dialogue socratique, quel est le role principal du tuteur ?**
   - A. Donner directement la reponse a l'eleve
   - B. Expliquer toute la matiere avant les exercices
   - C. Corriger immediatement chaque erreur
   - D. Poser des questions pour stimuler la reflexion de l'eleve
   - **Bonne reponse : D**

5. **Dans la methode de classe inversee utilisee au Centre Reed, quel est le role principal de l'eleve durant la seance de tutorat ?**
   - A. Ecouter les explications du tuteur avant de faire les exercices
   - B. Reviser la matiere et repondre aux questions du tuteur
   - C. Observer les methodes du tuteur pour les reproduire plus tard
